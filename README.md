# Karri Mobile

Karri Mobile is a mobile-first peer-to-peer shipping platform for trusted cross-border package delivery.

The first users are the East African diaspora.

This repository is a fresh start. It does not reuse the previous Karri monorepo, Turborepo setup, Azure App Services deployment, Prisma pipeline, or old backend architecture.

## Current phase

Foundation and documentation-first setup.

## Core principle

Trust is the product.

Every product, design, engineering, and operations decision should improve at least one of:

- Trust
- Simplicity
- Transparency
- Reliability
- Delight

## Preferred stack

- Expo React Native
- TypeScript
- Expo Router
- Firebase Authentication
- Cloud Firestore
- Cloud Functions
- Cloud Storage
- Firebase Cloud Messaging
- Firebase App Check
- EAS Build
- EAS Update
- MkDocs Material for documentation

## Documentation-first workflow

No feature begins with code.

Every feature follows:

1. Vision
2. Documentation
3. Architecture
4. Design
5. Engineering review
6. Implementation
7. Testing
8. Deployment
