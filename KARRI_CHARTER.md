# Karri Charter

## Mission

Karri helps people move packages across borders through trusted travelers, clear custody tracking, and transparent communication.

## Vision

Build the world's most trusted mobile-first peer-to-peer cross-border shipping platform, beginning with the East African diaspora and expanding globally over time.

## What Karri must protect

- User trust
- Package safety
- Identity confidence
- Custody history
- Clear expectations
- Community reputation

## Product promise

Karri should make every sender feel:

"I know who has my package, where it is, what happens next, and what proof exists."

## MVP focus

The first version must do one thing exceptionally well:

1. Sign in
2. Create shipment or trip
3. Match
4. Book
5. Track custody
6. Complete delivery
7. Review

## Non-negotiables

- Documentation before code
- Server-side business operations
- Event-driven platform behavior
- Immutable custody history
- Centralized notification service
- Remote configuration for changing business rules
- Security by default
